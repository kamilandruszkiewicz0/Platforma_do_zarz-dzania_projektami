<h1>Do włączenia projektu będzie potrzebny:</h1>
<ul>
  <li>
    Edytor kodu (np. VSCode)
  </li>
  <li>
    Node.js
  </li>
</ul>
<h2>Następnym krokiem jest pobranie projektu<br/>
    Należy utworzyć nowy terminal i wprowadzić w nim następujące komendy:</h2>
<ul>
  <li>
    cd [folder_projektu]
  </li>
  <li>
    npm install
  </li>
  <li>
    npm run dev
  </li>
</ul>
